# Brainrot-Delta
Mini hub script for Steal a Brainrot on Delta (Android
# Steal a Brainrot Delta Hub (Android)

Это мини‑скрипт‑хаб для игры **Steal a Brainrot** под **Delta Executor на Android**.  
Через этот скрипт ты можешь включать разные функции (скорость, ESP, автозабор брейнротов и т.д.) прямо в меню.

---

## 🔧 Функции (Features)

- **Speed**  
  - Увеличение WalkSpeed и JumpPower.  
- **Inf Jump / Infinity Jump**  
  - Бесконечный прыжок (удержание Space).  
- **Desync**  
  - Лёгкий десинк (Wiggle / High Desync).  
- **ESP**  
  - ESP для самых крутых брейнротов (Best Brainrot).  
  - ESP для баз и времени (Base Time).  
- **Auto Grab**  
  - Auto Grab All — забирает все брейнроты рядом.  
  - Auto Grab To Best — ищет и забирает “лучший” брейнрот.  
- **Auto Buy Brainrot**  
  - Автопокупка брейнротов.  
- **Auto Duel / Duel Finder**  
  - Авто‑дуэли и поиск дуэлей.  
- **Trade Finder**  
  - Поиск игроков для обмена.  
- **Auto Kick After Steal**  
  - Автоматический кик после кражи брейнрота (защита от бана).  
- **Lagger Speed**  
  - Скорость, похожая на лаг‑хак.  
- **TP To Best Brainrot**  
  - Телепорт к самому лучшему брейнроту на сервере.  
- **Anti‑Exploits**  
  - Anti Ragdoll  
  - Anti Bee Disco  
  - Anti Sentry  
- **Semi‑Invisible / Instant Clone**  
  - Полу‑невидимость.  
  - Instant Clone (клонированный персонаж).  
- **Server Hopper / Auto Joiner**  
  - Через внешние сайты‑хабы (Sammy, Lennranda, stealer.fun и др.),  
    сам скрипт не выбирает серверы по крутости брейнрота.
